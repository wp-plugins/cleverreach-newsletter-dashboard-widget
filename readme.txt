Extend your WordPress dashboard with latest statistics from your CleverReach Newsletter campaign. 

== Description ==
Connect your WordPress Dashboard to your [CleverReach Newsletter](http://www.cleverreach.com/?rk=74089ojcjrsrb) account.

This plugins shows latest statistics from your campaigns as dashboard widget.

See [screenshots](https://wordpress.org/plugins/cleverreach-newsletter-dashboard-widget/screenshots/) for more details.

[More information](http://etzelstorfer.com/cleverreach-wordpress-dashboard-widget/) avalaible in German

= Integrate CleverReach in your Contact form =
Have a look at the [NinjaForms CleverReach plugin](http://etzelstorfer.com/ninja-forms-cleverreach/) to send any contact form submissions directly to your CleverReach lists.

= Free Beautiful HTML Mails for your WordPress site =
With [WP HTML Mail](http://wp-html-mail.com/) you can create beautiful responsive mail templates for your WordPress mails without any coding.

== Installation ==
Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.


== Frequently Asked Questions ==

= How can I use a different API key? =

Disable an reenable the plugin to reset the configuration.
